# 🎃 ハロウィンジャー試作

“やり方”は教えてもらえない——動かして観察し、お化けの**心の開き方**を推理する小さなパズルです。
外部依存はありません。`index.html` をブラウザで開くだけで遊べます（GitHub Pages でもそのまま動きます）。

## 遊び方

1. **動かす（=記録）**
   方向キー / **WASD** で主人公を動かすと、その軌跡が「計画」に記録されます。
   動かすのは**記録であって即実行ではありません**。記録した動きはタイムラインで先読みできます。
   - `←/A` 左へ・`→/D` 右へ（1回=1マス、押しっぱなしで連続移動）
   - `↑/W/Space` ジャンプ（穴を越える）
   - `↓/S` しゃがむ・`Z` 待つ
   - `Backspace` ひとつ戻して録り直し

2. **再生して観察する**
   **▶ 再生**で計画どおりに実行します（先読み=再生は必ず一致）。
   お化けが**どう反応したか**が 📓 **ヒーローノート**に「事実」として自動で記録されます。

3. **推理する**
   ノートにたまる事実（「はなれてしゃがむと安心」「近づくと後ずさる」など）を読み比べて、
   この子に何が効くかを推理します。**事実が3つ**たまると 🔎 **仮説**（「〇〇かも？」）がひらめきます。

4. **記録の増やし方**
   いろいろ動かして **▶ を最後まで再生する**たびに、観察した事実が増えます。
   増えた瞬間は **📝NEW** で光ってお知らせします。**ちがう動きほど、ちがう発見**。

## お化けの読み方

色と仕草で気持ちを読みます（数値ゲージは出ません）。

- 青く震える＝こわがり／紫で目を逸らす＝警戒／気にし始める＝興味
- 桃色で寄ってくる＝安心／灰色で固まる＝心を閉ざした

## 環境ギミック

- 🌿 **物陰**でしゃがむと視線を切って、過敏な子の眼前でも怖がらせずに近づけます。
- 🍎 **思い出の品**のうち“好きだった物”のそばでしゃがむと、早く心を開きます（効いている時は品が ♪ と光ります）。
- どちらの反応もノートに事実として記録され、推理の材料になります（正解は教えません）。

## 困ったら

- **「お手本をなぞる」** で必ずクリアできます。
- **「会話をもう一度」** で入口（この子の気持ち）を読み直せます。
- 攻撃はしません。**間合い**を読んで**信頼**を育て、隣り合えたらクリアです。

楽しんでください！🎃
